:::tip[Use MikroCanvas Online]
Open [canvas.mikrosuite.com](https://canvas.mikrosuite.com) to use MikroCanvas online for free. It runs directly in the browser over HTTPS, requires no account, and stores boards privately in browser storage for that site.
:::

## Run Your Own Copy

Download the static app archive and serve it locally:

```bash
curl -sSL -o mikrocanvas.zip https://releases.mikrosuite.com/mikrocanvas_latest.zip
unzip mikrocanvas.zip -d mikrocanvas
cd mikrocanvas/*
npx http-server . -a 127.0.0.1 -p 8000 -c-1
```

Open `http://127.0.0.1:8000`.
