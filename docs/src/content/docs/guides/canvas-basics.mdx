---
title: Working on the Canvas
description: Core canvas workflows.
---

MikroCanvas opens directly into the board surface. The top bar manages boards and file actions, the bottom dock changes drawing tools, and the zoom controls keep the board framed.

## Tools

- Select moves, resizes, duplicates, deletes, and edits existing elements.
- Hand pans the board without changing elements.
- Sticky note, text, the shape selector, arrow, pen, and comment create new content. The shape selector remembers the last chosen shape and includes basic, polygon, and flowchart-friendly objects like capsules, documents, databases, trapezoids, octagons, and chevrons.
- The style toolbar appears when something is selected and includes fill, stroke, text color, line style, font size, ordering, duplication, and deletion.
- The lower-right controls include history, grid visibility, and zoom.

## Editing

Double-click an element, press Enter on a selected element, or use the edit button in the style toolbar to edit text. Text-capable elements are saved as soon as they are placed, so a quick browser reload will not drop a newly created shape or note.

## Keyboard shortcuts

Open the command palette with Cmd/Ctrl+K to find tools, board actions, exports, view controls, and selection commands. MikroCanvas also supports common editing shortcuts: select all, copy, cut, paste, undo, redo, duplicate, delete, zoom in, zoom out, and one-key tool switching.
